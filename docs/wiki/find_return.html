Find & Return

Search for goal flags in a procedurally-generated maze with destructible walls. When an agent finds a flag, it is rewarded and teleported to a new random position.

Game Rules

Map Generation

The map is generated using fractal noise. Tiles above a configurable threshold (mapgen_threshold, default 0.3) become destructible walls. Decorative tiles fill the remaining empty space. The map is padded with permanent walls for observation slicing.

Flags

A configurable number of flags (num_flags) are placed on random empty tiles at reset. Flags remain on the map — when an agent walks onto a flag tile, the agent is rewarded and teleported to a random empty position. The flag itself stays in place for other agents.

Movement & Digging

  • Agents can move in the 4 cardinal directions.
  • Moving into a wall or destructible wall blocks movement.
  • When an agent attempts to move into a destructible wall, the wall is destroyed and the agent enters a digging timeout (digging_timeout steps, default 5). During this timeout the agent cannot act.

Actions Used

IDAction
0Move up
1Move right
2Move down
3Move left

Stay, primary action, and dig are masked off. Digging happens implicitly when moving into a destructible wall.

Rewards

  • +treasure_reward (default 1.0) when an agent steps onto a flag tile.

Termination

Episode ends after length steps (set at environment creation).

Observations

Standard 4-channel format. Only the tile channel is meaningfully used — direction, team, and health are all zeros.
