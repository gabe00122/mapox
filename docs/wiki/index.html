MAPOX Environments

A collection of JAX-native, multi-agent, partially-observable gridworld environments with a shared observation/action format.

Shared Interface

All environments are functional (state in, state out), designed for jax.jit/jax.vmap, and expose an action_mask for per-environment action subsets.

env.reset(rng_key) -> (state, timestep)
env.step(state, actions, rng_key) -> (state, timestep)

Action Space

All environments share DiscreteActionSpec(n=7). Not every env uses every action — always consult timestep.action_mask.

IDAction
0Move up
1Move right
2Move down
3Move left
4Stay
5Primary action
6Dig

Observation Format

Each agent receives a local crop: (view_w, view_h, 4) int8 with channels:

ChannelMeaningRange
0Tile ID (terrain + agents)0–18
1Direction0=none, 1–4=cardinal
2Team ID0=none, 1=red, 2=blue
3Health0–2

Environments

Find & Return
Search for goal flags in a procedurally-generated maze. When an agent finds a flag it is rewarded and respawned elsewhere.
Solo / Cooperative
Scouts
Two roles: Harvesters unlock treasure tiles, Scouts collect unlocked treasures. Asymmetric cooperative play.
Cooperative Asymmetric Roles
Traveling Salesman
Collect scattered flags in any order. Each agent tracks which flags it has visited; flags reset after all are collected.
Solo / Cooperative
King of the Hill
Two-team competitive combat with knights and archers, destructible walls, control points, and team-shared rewards.
Competitive Two Teams
Soccer
Two teams play on an open field with a pushable ball and goals at opposite ends. Simultaneous actions with randomized execution order.
Competitive Two Teams
Stealth
Prey vs predator: sneakers gather food and hide in grass while chasers hunt them. Asymmetric predator-prey dynamics.
Mixed Motive Asymmetric Roles
